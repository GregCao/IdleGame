# 放置游戏 - 数值平衡设计

## 1. 核心数值公式

### 1.1 玩家属性公式

```
攻击力 = 10 + (等级 - 1) × 5
生命值 = 100 + (等级 - 1) × 20
暴击率 = 10% (固定)
暴击伤害 = 150% (固定)

升级消耗金币 = 等级 × 100
例: Lv1→Lv2 = 200金, Lv10→Lv11 = 1000金, Lv100→Lv101 = 10000金
```

### 1.2 怪物属性公式

```
怪物生命 = 玩家等级 × 50 × 波次系数
怪物攻击 = 玩家等级 × 2 × 波次系数
金币奖励 = 玩家等级 × 20 × 波次系数

波次系数 = 1 + (波次 × 0.3)
例: 波次1 = 1.3, 波次10 = 4.0, 波次50 = 16.0
```

### 1.3 离线收益公式

```
每秒金币 = 攻击力 × 0.5
离线金币 = 每秒金币 × 离线秒数
离线上限 = 8小时 (28800秒)
```

### 1.4 装备强化公式

```
强化消耗 = 基础消耗 × (1.5 ^ 强化等级)
属性加成 = 基础加成 × (1 + 强化等级 × 0.1)

例: 武器基础消耗100金, +10强化后 = 100 × 1.5^10 = 5757金
```

## 10. 调优建议

### 10.1 如果玩家升级太快
- 增加升级金币消耗系数
- 公式调整为: `cost = level × 100 × 1.2^(level-1)`

### 10.2 如果玩家卡关严重
- 降低波次怪物属性系数
- 调整为: `波次系数 = 1 + (波次 × 0.2)`

### 10.3 如果付费转化低
- 增加首充礼包价值
- 降低月卡价格
- 增加限时折扣活动

### 10.4 如果广告收入低
- 增加激励视频场景
- 优化广告展示时机（关卡间/死亡时）
- 针对T1市场优化
